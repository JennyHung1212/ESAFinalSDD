Product Family and Release Schedule

Description

Overall, we focused our efforts on reusing the same game mechanics and rules to expand on our original product.

Branching out from the first version of the game, we can offer greater content variety by releasing expansion packs. These will offer the same game, with the same mechanics and production process, just different or added content about Singapore.

Likewise, further down the line, we can entirely switch up the content by introducing other countries instead using the same game rules and artifacts.

Still using the same game mechanics, we can release a digitized version, and be able to update content for players without them having to buy an additional game. This will be more environmentally friendly compared to the addition of physical playing card.
